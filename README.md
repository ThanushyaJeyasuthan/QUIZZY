**QUIZZY**
QUIZZY is a modern, responsive educational platform designed to make learning fun and interactive. This project was developed as part of the COM 2303 - Web Design and Technologies course at Rajarata University of Sri Lanka.

**Theme**
The theme of QUIZZY is "Learning through Engagement." 

**Visual Identity** The app uses a calming mint-green and dark-blue palette to create a focused yet welcoming environment for students.


**Key Features**
  Dual Access Modes: Users can choose to Login/Register for a personalized experience or Play as Guest to start a session immediately.
  Dynamic Content Carousel: An interactive category slider (Maths, GK, and IT) that allows users to browse subjects before starting.
  Dynamic Question Logic: Questions change dynamically so that repeating a category provides a fresh challenge every time.
  Subject Categories: Specialized content for ICT, General Knowledge, and Mathematics.
  Smart Results Page:
    Interactive Scoring: Automatic calculation of correct vs. incorrect answers.
    Feedback System: Users can provide suggestions and star ratings directly on the results page.
    "Did You Know?" Section: Displays interesting facts tailored to the user's selected category to encourage continuous learning.

**Usage Instructions**
1. Prerequisites
  No server is required for this version. You only need a modern web browser (Chrome, Firefox, or Edge).
2. How to Run
  Download: Clone or download this repository from the main branch.

Launch: Open index.html in your browser.

**Quiz Flow:**
  Click Login/Register or Play as Guest on the landing page.
  Use the carousel to pick a Category (IT, GK, or Maths).
  Complete the quiz and review your Results.
  Read the "Did You Know" facts and leave a Rating.

**Folder Structure**
  index.html - The landing page shown in the screenshot.
  category.html - Subject selection interface.
  quiz.html - The dynamic quiz engine.
  results.html - Results display with feedback and "Did You Know" sections.
  css/ & js/ - Frontend styling and interactive logic.
